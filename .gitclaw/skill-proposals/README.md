# GitClaw Skill Proposals

This directory is reserved for reviewed skill proposal drafts.

Use one proposal per directory:

```text
.gitclaw/skill-proposals/<skill-name>/PROPOSAL.md
```

Proposal files are inert review artifacts. GitClaw may inventory and risk-audit
them, but it does not auto-apply proposals, create active skills, run support
scripts, install dependencies, or mutate `.gitclaw/SKILLS`.
